# GettingandCleaningDataCourseProject
The purpose of this project is to demonstrate your ability to collect, work with, and clean a data set. It includes the instructions and scripts to perform the analysis on the Human Activity Recognition database.

## Dataset

* **Dataset**: Human Activity Recognition Using Smartphones
* **Source**: [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones)

## Files

* `CodeBook.md`: A code book that describes the variables, the data, and any transformations or work that was performed to clean up the data.
* `run_analysis.R`: The R script that performs the data preparation as described in the course projectâ€™s definition.
* `final_tidy_data.txt`: The exported final data after going through all the sequences described above.
